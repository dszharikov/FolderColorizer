# Contributing

Thank you for helping improve Folder Colorizer.

1. Create a focused branch from `main`.
2. Run `dotnet test FolderColorizer.slnx --configuration Release`.
3. Keep changes small and preserve existing `desktop.ini` data.
4. Open a pull request with a concise explanation and screenshots for UI changes.

The project follows the existing `.editorconfig` and treats compiler warnings as
errors.
